## 📋 Descripción

Aplicación orientada a encargados de negocio que permite registrar, consultar y gestionar pedidos del día de forma sencilla. El sistema busca reemplazar el control manual de pedidos, reduciendo errores y mejorando la visibilidad del flujo de trabajo.

---

## 🗂️ Backlog del Producto

| ID | Tipo | Historia de Usuario | Sprint | Puntos | Prioridad |
|---|---|---|---|---|---|
| US-01 | Registro de pedido | Como encargado del negocio, quiero registrar un pedido con nombre del cliente, productos, cantidades y total, para no perder pedidos y tener un control claro. | Sprint 1 | 5 | Alta |
| US-02 | Lista de pedidos activos | Como encargado, quiero ver todos los pedidos del día en una lista, para saber cuáles están pendientes y cuáles en proceso. | Sprint 1 | 3 | Alta |
| US-03 | Cambio de estado | Como encargado, quiero cambiar el estado de un pedido entre Pendiente, En proceso y Entregado, para mantener actualizado el flujo de trabajo. | Sprint 1 | 3 | Alta |
| US-04 | Historial de cliente | Como encargado, quiero buscar un cliente y ver sus pedidos anteriores, para identificar clientes frecuentes y sus preferencias. | Sprint 2 | 5 | Media |
| US-05 | Resumen del día | Como encargado, quiero ver un resumen al final del día con el número de pedidos atendidos y el total cobrado, para evaluar el desempeño del negocio. | Sprint 2 | 3 | Media |
| US-06 | Eliminar pedido | Como encargado, quiero poder eliminar un pedido registrado por error, para mantener el listado limpio. | Sprint 2 | 2 | Baja |
| US-07 | Exportar pedidos | Como encargado, quiero exportar los pedidos del día como archivo de texto o CSV, para tener un respaldo simple. | Sprint 2 | 3 | Baja |

## 🚀 Planificación de Sprints

### Sprint 1 — Funcionalidades Core
Enfocado en las historias de mayor prioridad para tener un MVP funcional.

- US-01 · Registro de pedido (5 pts)
- US-02 · Lista de pedidos activos (3 pts)
- US-03 · Cambio de estado (3 pts)

**Total: 11 puntos**

### Sprint 2 — Funcionalidades Extendidas
Agrega visibilidad histórica, reportes y mantenimiento del sistema.

- US-04 · Historial de cliente (5 pts)
- US-05 · Resumen del día (3 pts)
- US-06 · Eliminar pedido (2 pts)
- US-07 · Exportar pedidos (3 pts)

**Total: 13 puntos**
